As a short college project we were tasked with designing a poster using geometric shapes, lines and other vectors along with a model image to create an 80's style theme. Each of these elements were supplied to us.

Design and creation.

Being one of my first real projects using illistrator and Photoshop my design process consisted mainly of trial and error along with experimentation. Taking inspiration from toher 80's art designs I was able to layer the supplied vector assets around the model image and proceeded to use Photoshop to edit the colour balance and other enhancements.
